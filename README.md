*UKL API Project*

Proyek ini adalah REST API sederhana berbasis **Node.js**, **Express**, dan **Prisma** yang dibuat untuk keperluan **UKL (Uji Kompetensi Keahlian)**.  
Project ini dirancang agar mudah dipahami oleh pelajar yang sedang belajar membangun API menggunakan JavaScript dan Express.

---

*Endpoint Utama*

| Route | Method | Deskripsi |
|-------|---------|-----------|
| `/user` | GET | Menampilkan data user |
| `/user/:id` | GET | Menampilkan user berdasarkan ID |
| `/attendance` | GET | Menampilkan data kehadiran |
| `/auth` | POST | Login atau autentikasi pengguna |

> Semua endpoint dapat diuji menggunakan **Postman** dengan base URL:  
> `http://localhost:8000`

---

Proyek **UKL API** ini membantu memahami dasar-dasar dalam membangun **REST API** menggunakan **Node.js** dan **Express**.  
Dengan struktur yang sederhana, proyek ini cocok untuk pelajar yang ingin belajar konsep **routing**, **controller**, dan **integrasi database** melalui **Prisma**.  
Dari sini, kamu bisa mengembangkan fitur-fitur lanjutan seperti autentikasi, validasi data, dan koneksi ke database nyata.

>  Intinya, proyek ini adalah langkah awal yang kuat untuk memahami bagaimana server bekerja di dunia backend development.
